# CONTEXT.md — Regras Obrigatórias de Contexto

## REGRA 1 — Skill de Humanização (Copy e Texto)

Antes de escrever qualquer texto, copy, descrição, legenda, headline ou conteúdo de interface neste projeto, CARREGAR e seguir a skill:

```
~/.claude/skills/humanizer/
```

Esta skill define o tom, voz e identidade textual do projeto UGC Prompt Generator. Todo copy deve respeitar as diretrizes dessa skill antes de ser escrito.

---

## REGRA 2 — Skill de Design Frontend (Interface)

Antes de qualquer alteração de interface, componente, estilo ou layout neste projeto, CARREGAR e seguir a skill:

```
.claude/skills/frontend-design/SKILL.md
```

Esta skill define a identidade visual dark/premium do projeto: paleta de cores, tipografia, componentes, animações e regras inegociáveis de design. Nenhuma alteração de UI deve ser feita sem consultar esse arquivo primeiro.
